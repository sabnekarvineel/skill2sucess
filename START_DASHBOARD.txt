Problem: Dashboard not loading
Solution:
  1. Make sure you're logged in
  2. Go to "My Skills" and add at least one skill
  3. Refresh the page
  4. Check browser console for errors

Problem: Recommendations not updating
Solution:
  1. Go to "My Skills" and save again
  2. Wait 2-3 seconds
  3. Refresh the browser (Ctrl+R or Cmd+R)

Problem: Links not working
Solution:
  1. Allow pop-ups in your browser
  2. Right-click link and "Open in new tab"
  3. Check internet connection

Problem: Mobile view looks odd
Solution:
  1. Try scrolling - the dashboard is responsive
  2. Rotate device to landscape
  3. Clear browser cache
  4. Try a different browser
