### Skill Matching Algorithm

**Step 1: Get User Skills**
```
User.skills = {
  programming: ["Python", "SQL"],
  databases: ["MySQL"],
  frameworks: ["Pandas"],
  tools: ["Excel", "Tableau"],
  softSkills: ["Problem Solving"]
}
```

**Step 2: Get Required Skills**
```
DataAnalyst.requiredSkills = {
  programming: ["Python", "SQL", "R"],
  databases: ["MySQL", "PostgreSQL", "MongoDB"],
  frameworks: ["Excel", "Tableau", "Power BI"],
  tools: ["Jupyter", "Git", "Google Analytics"]
}
```

**Step 3: Find Matches**
```
Flattened User: [Python, SQL, MySQL, Pandas, Excel, Tableau, Problem Solving]
Flattened Required: [Python, SQL, R, MySQL, PostgreSQL, MongoDB, Excel, Tableau, Power BI, Jupyter, Git, Google Analytics]

Matches: [Python, SQL, MySQL, Excel, Tableau] = 5 skills
Total Required: 12 skills
```

**Step 4: Calculate Score**
```
Score = (5 / 12) Ã— 100 = 41.67%
Category = "Developing"
```
